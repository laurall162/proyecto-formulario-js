# Proyecto formulario JavaScript

Este proyecto consiste en una página web sobre museos para visitar con niños en Asturias.

La web incluye:

- Página principal con información sobre museos.
- Imágenes y enlaces a los museos.
- Un formulario de creación de cuenta.
- Validación del formulario realizada con JavaScript.

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript
